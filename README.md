📘 Reading Habit — 習慣化読書アプリ
「読書体験を、ただの消費から一生の資産へ。」 > 短時間の読書を確実にアウトプットと習慣に結びつける、社会人のための読書支援ツール。

🚀 Live Demo を体験する (Firebase Hosting を利用して公開中)

🎯 解決したい課題
時間の不足: 忙しい日常の中で、まとまった読書時間が確保しにくい。

定着の欠如: 読んでも内容を忘れがちで、実際の行動に活かしにくい。

継続の難しさ: 一人ではモチベーションを維持し、習慣化するのが難しい。

✨ 主な機能（現在の実装状況）
現在、以下のコア機能を Web アプリとして実装・展開しています。

📚 本の登録・管理: 読みたい本や現在読んでいる本を、直感的なカード形式で整理できます。

⏱ 読書タイマー: セッションごとの時間を計測し、高い集中力を維持した読書をサポートします。

🧠 クイック・リフレクション: 読書直後の「気づき」や「思考」を、最小限の手間で記録できます。

🔥 Firebase 連携: Google ログインによる認証とデータ同期により、デバイスを問わず記録が可能です。

🚀 技術スタック
シンプルかつ堅牢な構成で開発しています。

Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Backend: Firebase (Authentication / Cloud Firestore)

Hosting: Firebase Hosting

🏗 今後のロードマップ（将来の拡張）
🤖 AI フィードバック: 記録した気づきに対し、AIが学びを深めるアドバイスを提示。

📱 モバイルアプリ化: Flutter を用い、iOS/Android 両対応へ拡張。

📊 読書統計の可視化: 継続日数や読書量をグラフ化し、達成感を可視化。
