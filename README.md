📘 Reading Habit — 習慣化読書アプリ
忙しい社会人が、短時間の読書を確実にアウトプットと習慣につなげるための読書支援ツールです。

🎯 解決したい課題
「読書する時間が確保しにくい」

「読んでも内容を忘れがちで、行動に活かしにくい」

「一人だと習慣として定着しにくい」

✨ 主な機能（現在の実装状況）
現在、以下のコア機能を Web アプリとして実装・展開しています。

本の登録・管理: 読みたい本や現在読んでいる本を直感的なカード形式で管理。

読書タイマー（セッション記録）: 読書時間を計測し、集中力を維持。

クイック・リフレクション: 読書直後に、その時の「気づき」や「思考」を最小限の手間で記録。

Firebase 連携: データの保存と認証（Google ログイン）により、どのデバイスからでも記録が可能。

🚀 技術スタック
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Backend: Firebase (Authentication / Firestore)

Hosting: Firebase Hosting

🏗 今後のロードマップ（将来の拡張）
AIによる振り返りフィードバック: 記録した気づきに対してAIがアドバイスを行い、学びを深める支援。

モバイルアプリ化: Flutter を用いた iOS/Android 両対応。

読書統計の可視化: 継続日数や読書量のグラフ表示。
