📘 Reading Habit — 習慣化読書アプリ
✨ 概要

忙しい社会人が 短い時間でも読書を習慣化 できるように設計された無料アプリです。
読書時間そのものよりも、**その都度感じた「気づき」や「思考」を記録することで、読書体験を行動につなげます。

🎯 課題とペルソナ
